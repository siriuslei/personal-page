# Sirius Zhou (Lei Zhou) - Personal Portfolio Website

A premium, responsive, single-page personal showcase website for **Lei Zhou (周磊)**. It presents his dual identities as an Industrial Designer / Project Manager and a Bilingual VFX Content Creator.

## Features
- **Bento Grid Layout**: A highly structured 12-column responsive layout.
- **Dual-Accent Design**: Teal/Cyan for industrial engineering precision, and Violet/Purple for creative visual effects.
- **Awwwards-style Hover Glow**: Real-time mouse tracking radial gradients on card backgrounds.
- **Bilingual Engine**: Fast, localized text rendering with instant EN/中文 toggling and language selection persistence via `localStorage`.
- **Scroll Animations**: Smooth entry scroll reveal transitions using `IntersectionObserver`.

## Local Development
To serve and preview the website locally, run:
```bash
python3 -m http.server 8000
```
Then open [http://localhost:8000](http://localhost:8000) in your browser.
